## Installation

1. Clone the repository:

   ```
   git clone https://github.com/shahyash1136/python.git
   ```

2. Navigate to the project directory:

   ```
   cd python
   ```

3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the main script:

```
python main.py
```
